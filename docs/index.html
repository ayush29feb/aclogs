Setup
01
Clone and open in Claude Code

Clone the repo onto your Mac and open it in Claude Code.

git clone https://github.com/ayush29feb/gym && cd gym && claude
02
Ask Claude to set up

Claude runs the setup script, installs all dependencies, builds the server, and creates your database config.

"Set up AC Logs for the first time"
03
Import your workouts

Add your workout markdown files to the workouts/ directory and run the import script to populate the database.

cd cli && uv run python import_workouts.py
04
Open on your phone via Tailscale

Install Tailscale on your Mac and phone. Ask Claude for the URL and bookmark it.

"Start the servers and give me the Tailscale URL"
